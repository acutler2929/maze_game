# maze_game
Game Name: Maze Escape 

Game Idea 

Our game is a simple 2D maze game. The player starts at the beginning of the maze and tries to find the exit. There are many different paths in the maze, but only one path will lead to the correct exit. Some paths are dead ends, and some have bombs. If the player touches a bomb, the game will restart from the beginning. 

The player can also collect coins while moving in the maze. Coins are optional but collecting them will increase the score. The goal is to escape the maze and collect as many coins as possible without touching any bombs. 

Game Objective 

Find the correct way out of the maze. 

Collect coins to get more points. 

Do not touch bombs. 

Reach the exit to win the game. 

Game Features 

2D top-down maze 

Player movement using arrow keys 

Coins to collect 

Bombs that restart the game 

Score system 

Win screen 

Restart after losing 

Multiple paths inside the maze 

Only one correct exit path
